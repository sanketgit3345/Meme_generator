# Meme Generator

A React-based web application that allows users to generate memes by selecting images and adding custom text.

## Features
- Upload or select an image
- Add top and bottom text to create a meme
- Adjust text position and size
- Download the generated meme
- Responsive design for mobile and desktop users

## Technologies Used
- React.js
- HTML & CSS
- JavaScript
- Canvas API

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/sanketgit3345/Meme-Generator.git
   ```
2. Navigate to the project folder:
   ```sh
   cd Meme-Generator
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
